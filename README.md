# strong-plugin-parent

## 介绍

一个建单的Spring MVC工程增强插件的父类定义



## 组件介绍

### `mybatis-secure-plugin`

Mybatis增强组件，支持数据持久化时自动加密、查询结果自动解密、查询结果自动脱敏。具体说明查看`mybatis-plugin/readme.md`



### `json-data-plugin`

Json数据增强组件，数据序列化时脱敏。具体说明查看`json-data-plugin/readme.md`
